### Technical Specifications

- **Blockchain**: Solana (initial), Ethereum L2s (2027+)
- **Token Standard**: SPL Token
- **Total Supply**: 1,000,000,000 NEBA (fixed maximum)
- **Distribution Model**: Epoch-based (30-60 minute cycles)
- **Verification**: Optimistic rollup with oracle consensus
